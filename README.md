# FileReflector

A simple web interface to select which files should be synchronized.

## Why this exists

- Couldn't find a simple tool that just did the job, so I made one — one feature, focused and straightforward.
- Take advantage of well-known linux utilities - no reinventing the wheel.

## Uses

- Selective Media Syncing
Pick specific files from a Storage Server to sync to a Media Server, keep only what you want to watch.

- Media Production Pipelines
In video or audio production, sync only the required assets from a shared storage to a rendering or editing workstation.
